# Installing Past Releases of Rawi CLI

This topic describes how to install previous releases of the Rawi Command Line Interface (Rawi CLI) on supported operating systems. For information on the latest releases, see the [Rawi GitHub Releases](https://github.com/MKAbuMattar/rawi/releases).

## System Requirements

- **Node.js**: 18.0.0 or higher
- **Package Manager**: npm, pnpm, or yarn
- **Operating System**: Windows, macOS, or Linux

## Installation Instructions

<Tabs>
  <TabItem label="Linux / macOS" icon="linux">
    <Steps>
      1. **Choose your package manager and run the install command:**

         <Tabs>
           <TabItem label="npm" icon="npm">
             ```bash
             npm install -g rawi@<version>
             ```
           </TabItem>

           <TabItem label="pnpm" icon="pnpm">
             ```bash
             pnpm add -g rawi@<version>
             ```
           </TabItem>

           <TabItem label="yarn" icon="seti:yarn">
             ```bash
             yarn global add rawi@<version>
             ```
           </TabItem>
         </Tabs>

      2. **(Optional) Install a specific beta version:**

         ```bash
         npm install -g rawi@beta
         ```

      3. **(Optional) Install from GitHub (latest main branch):**

         ```bash
         npm install -g withrawi/rawi
         ```

      4. **Verify installation:**

         ```bash
         rawi --version
         rawi info
         ```
    </Steps>

  </TabItem>

  <TabItem label="Windows" icon="seti:windows">
    <Steps>
      1. **Open PowerShell or Command Prompt as Administrator.**
      2. **Choose your package manager and run the install command:**

         <Tabs>
           <TabItem label="npm" icon="npm">
             ```powershell
             npm install -g rawi@<version>
             ```
           </TabItem>

           <TabItem label="pnpm" icon="pnpm">
             ```powershell
             pnpm add -g rawi@<version>
             ```
           </TabItem>

           <TabItem label="yarn" icon="seti:yarn">
             ```powershell
             yarn global add rawi@<version>
             ```
           </TabItem>
         </Tabs>
      3. **Verify installation:**

         ```powershell
         rawi --version
         rawi info
         ```
    </Steps>

  </TabItem>
</Tabs>

## Notes

<Tabs>
  <TabItem label="npx" icon="npm">
    You can always use `npx` to run a specific version without installing globally:

    ```bash
    npx rawi@<version> ask "Hello, world!"
    ```

  </TabItem>

  <TabItem label="pnpm dlx" icon="pnpm">
    Or use `pnpm dlx`:

    ```bash
    pnpm dlx rawi@<version> configure
    ```

  </TabItem>
</Tabs>

For troubleshooting installation issues, see the [Troubleshooting Guide](./docs/troubleshooting.md).

## Version Management

<Tabs>
  <TabItem label="Semantic Versioning" icon="document">
    Rawi uses [semantic versioning](https://semver.org/):

    * `MAJOR.MINOR.PATCH`
      * **MAJOR**: Breaking changes
      * **MINOR**: New features (backwards compatible)
      * **PATCH**: Bug fixes (backwards compatible)
        For more details, see the [Release Process](./docs/development.md#release-process).

  </TabItem>
</Tabs>
