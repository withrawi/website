# Installation & Setup

Get Rawi up and running on your system in just a few minutes.

## System Requirements

Before installing Rawi, ensure your system meets these requirements:

- **Node.js**: Version 18.0.0 or higher
- **Operating System**: Windows, macOS, or Linux
- **Terminal**: Any modern terminal (Terminal.app, iTerm2, Windows Terminal, PowerShell, etc.)
- **Internet Connection**: Required for cloud AI providers (optional for local providers like Ollama)

### Check Your Node.js Version

```bash
node --version
```

If you need to install or update Node.js, visit [nodejs.org](https://nodejs.org/).

### Alternative Installation Methods

#### Using npx (No Installation)

Run Rawi without installing globally:

```bash
# Ask a question without installing
npx rawi ask "What is Docker?"

# Configure without installing
npx rawi configure
```

#### Development Installation

For contributing or development:

```bash
# Clone the repository
git clone https://github.com/mkabumattar/rawi.git
cd rawi

# Install dependencies
npm install

# Build the project
npm run build

# Link for global usage
npm link
```

## Post-Installation Setup

### Initial Configuration

After installation, configure your first AI provider:

```bash
# Interactive setup (recommended)
rawi configure

# Manual setup example
rawi configure --provider openai --model gpt-4o --api-key sk-your-key
```

### Shell Integration (Optional)

Add shell integration for enhanced experience:

#### Bash

Add to your `~/.bashrc`:

```bash
# Rawi shell integration
eval "$(rawi completion bash)"

# Useful aliases
alias ask='rawi ask'
alias chat='rawi ask --continue'
```

#### Zsh

Add to your `~/.zshrc`:

```bash
# Rawi shell integration
eval "$(rawi completion zsh)"

# Useful aliases
alias ask='rawi ask'
alias chat='rawi ask --continue'
```

#### PowerShell

Add to your PowerShell profile:

```powershell
# Rawi shell integration
Invoke-Expression (&rawi completion powershell)

# Useful aliases
Set-Alias ask rawi ask
```

## Troubleshooting Installation

### Common Issues

#### Permission Errors

If you encounter permission errors during global installation:

```bash
# Use sudo (Linux/macOS)
sudo npm install -g rawi

# Or configure npm to use a different directory
npm config set prefix '~/.npm-global'
export PATH=~/.npm-global/bin:$PATH
```

#### Node.js Version Issues

If you have an older Node.js version:

```bash
# Install Node Version Manager
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Install latest LTS Node.js
nvm install --lts
nvm use --lts
```

#### Command Not Found

If `rawi` command is not found after installation:

```bash
# Check if globally installed packages are in PATH
npm list -g --depth=0

# Add npm global directory to PATH
export PATH=$(npm config get prefix)/bin:$PATH

# For permanent solution, add to your shell profile
echo 'export PATH=$(npm config get prefix)/bin:$PATH' >> ~/.bashrc
source ~/.bashrc
```

#### Network Issues

If installation fails due to network issues:

```bash
# Use different registry
npm install -g rawi --registry https://registry.npmjs.org/

# Clear npm cache
npm cache clean --force

# Try with increased timeout
npm install -g rawi --timeout=60000
```

### Platform-Specific Issues

#### Windows

```powershell
# Set execution policy for PowerShell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

# Install using Chocolatey (alternative)
choco install nodejs
npm install -g rawi
```

#### macOS

```bash
# Install using Homebrew (alternative)
brew install node
npm install -g rawi

# Fix potential permission issues
sudo chown -R $(whoami) $(npm config get prefix)/{lib/node_modules,bin,share}
```

#### Linux

```bash
# Install Node.js on Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs

# Install Rawi
npm install -g rawi
```
