Rawi (راوي) is a developer-friendly AI-powered CLI tool that delivers clear answers, summaries, and analyses directly in your terminal. Inspired by the rich Jordanian storytelling tradition, Rawi transforms your command line into an intelligent assistant for your development workflow.

## <Icon name="rocket" color="var(--sl-color-text-accent)" /> Getting Started

To install Rawi (راوي), you can use your preferred package manager. Here are the commands for the most popular ones:

<Tabs>
  <TabItem label="pnpm" icon="pnpm">
    ```sh
    pnpm install -g rawi@latest
    ```
  </TabItem>

  <TabItem label="npm" icon="npm">
    ```sh
    npm install -g rawi@latest
    ```
  </TabItem>

  <TabItem label="yarn" icon="seti:yarn">
    ```sh
    yarn global add rawi@latest
    ```
  </TabItem>
</Tabs>
